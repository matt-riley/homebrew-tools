# matt-riley/tools

A Homebrew tap for tools built by [matt-riley](https://github.com/matt-riley).

> **Version policy:** versions listed here come directly from the tap formula files, not upstream release pages.

## Add the tap

```sh
brew tap matt-riley/tools
```

## Install

```sh
brew install matt-riley/tools/hopt
brew install matt-riley/tools/tui  # installs the `mattriley` binary
```
